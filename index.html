Typography Cheatsheet

Properties
Property Description Example
font-weight: Font Weight Used to change the impact of the font. font_weight: bold;
font-style: Font Style Used to change the style of the font. font_style: italics;
font-family: Font Family Used to sellect a font family for the text. font-family: "Times New Roman", Arial;
line-height: Line Height Used to add space between lines of text. line-height: 1.4;
word-spacing: Word Spacing Used to change the space between words. word-spacing: 4px;

Plumbing
Tag Name Description Example
link: Link Used to link to fonts that are hosted externaly.
@font-face: Font Face rule Used to host fonts localy. @font-face { font-family: 'Glegoo'; src: url('../fonts/Glegoo-Regular.ttf') format('truetype'); }
